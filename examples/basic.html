Basic usage

Basic usage of the React Autocomplete component looks like:

var options = [
  {id: 'banana', title: 'Banana'},
  {id: 'apple', title: 'Apple'},
  {id: 'pineapple', title: 'Pineapple'},
  {id: 'strawberry', title: 'Strawberry'},
]

<Autocomplete options={options} />
